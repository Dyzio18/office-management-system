# System management office

System to management law office. 

Created with:
- Spring
- Hibernate
- Maven
- MySQL
- Thymeleaf

## Preparation
1. Install Maven (and add to PATH)
2. Run MySQL database

## Run

1. mvn clean
2. mvn clean install
3. cd target
4. java -jar demo-0.0.1-SNAPSHOT.jar
5. open browser on port 8080

` http://localhost:8080/`
